## 来看 react

![](img/2020-12-29-18-27-05.png)

- virtual Dom, 并不会丢弃之前的数据，而是存储在memory。

- React高效的原因
  - 使用虚拟(virtual)DOM, 不总是直接操作页面真实DOM。
  - DOM Diffing算法, 最小化页面重绘。
